Citação Simples

Para fazer uma citação usamos a tag Q. Ex: Esforço + Disciplina = Sucesso!

Citação Completa

Para uma citação completa usamos a tag BLOCKQUOTE.

Ex: Lorem ipsum dolor sit amet consectetur adipisicing elit. A quod dignissimos iure fugit quis omnis laborum officia obcaecati dolorum veritatis consequatur ex provident nihil totam suscipit culpa, eos, tempore necessitatibus.

Abreviação

Para abreviar usamos a tag ABBR. Por exemplo a abreviação VASP

Texto invertido

Para criar um texto invertido basta usar a tag BDO. Igual essa frase que foi invertida
